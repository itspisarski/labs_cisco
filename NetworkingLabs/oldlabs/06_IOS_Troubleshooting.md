# Lab 06 – IOS Troubleshooting

**Objective:** Practice troubleshooting commands.

### 🧩 Topology
Router — Switch — PC

### ⚙️ Steps
1. Simulate misconfigured IP or disconnected cable.
2. Use:
   ```
   show ip int brief
   show run
   ping
   ```
3. Identify and fix issue.

### 🔍 Verification
- Issue resolved.
- Ping successful.
